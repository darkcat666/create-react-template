# CONTRIBUTION.md

This is how to use react projects for windows.

## usage

1. Go to root directory.

```bash
$cd ~
```

2. Check your location

```bash
$pwd
```

It should be as follows

```
Users/<yourname>
```

3. Make `tools` directory

```bash
$mkdir tools
```

4. Go to `tools` directory

```bash
$cd tools
```

5. install "pkg"

```bash
npm i -g pkg
```

6. install npm

```bash
npm install
```
